# MACHINE-LEARNING-LAB---5

This experiment focuses on implementing KNN and Decision Tree classification algorithms. KNN is applied to predict tumor diagnosis using the Breast Cancer Dataset. The target variable is Diagnosis (Benign / Malignant). Selected features include radius, texture, perimeter, area, and smoothness. Data preprocessing and label encoding are performed before training. Feature scaling is applied since KNN is distance-based. Different values of K are tested to analyze model performance. Accuracy, Precision, Recall, and F1 Score are used for evaluation. Confusion matrix and accuracy vs K plots are generated for visualization.

Decision Tree classification is implemented on the Loan Prediction Dataset. The target variable is Loan Status (Approved / Rejected). Missing values are handled and categorical variables are encoded. The dataset is split into training and testing sets. Tree depth is adjusted to compare shallow and deep trees. Feature importance and tree structure plots are analyzed. The experiment demonstrates model comparison and overfitting detection
