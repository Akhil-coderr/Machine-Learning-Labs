# MACHINE-LEARNING-LAB--4

This experiment implements Naive Bayes classification algorithms. Multinomial Naïve Bayes is used to classify SMS messages as Spam or Ham using the SMS Spam Collection Dataset. Text preprocessing and count vectorization are applied before training the model. Model performance is evaluated using Accuracy, Precision, Recall, and F1 Score. Gaussian Naïve Bayes is applied on the Iris Dataset for flower classification. Feature scaling and train-test splitting are performed before model training. Results are visualized using confusion matrix and decision boundary plots. The experiment demonstrates the effectiveness of Naïve Bayes for both text and numerical data.
